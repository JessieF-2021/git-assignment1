QUESTION: List all tags under html and add their description

TEXT TAGS

#h1 - h6: These are heading tags which define the heading of an HTML document from level 1 to level 6.
#p: This is known an the paragrapgh tag which is used to define a paragrapgh.
#strong: This tag is used to bolden a text thereby laying emphasis on that text.
#em: This tag is used to render or display a text in italics.
#abbr: The abbreviation tag defines the abbreviation for a longer word.
#var: The variable tag is used to define a variable.

SELF CLOSING TAGS

#br: The line break tag is used to break a line of code.
#hr: The horizontal rule tag is used to put a line across a webpage.

LINK TAGS

#a: The anchor tag is used to create a hyperlink.
#base: The base tag defines the URL for all relative URL within the document.

IMAGE AND OBJECT TAGS

#img: The image tag allows you add an image to your document.
#area: The area tag defines the area of an image map.
#param: The parameter tag defines the parameter of an object element.
#map: The map tag defines the map with active areas.
#object: The object tag is used to define an object in a document.

OTHER HTML TAGS

#main: The main tag defines the main content of an HTML document.
#aside: The aside tag defines content aside the main content and is normally represented as a sidebar.
#div: The division tag defines a section or division of HTML document.
#section: The section tag defines the generic section for a document.
#footer: The footer tag defines the footer of an HTML document.
#figure: The figure tag defines the self-contained content.
